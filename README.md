# KidWay Business Website

## Overview
KidWay Business Website is the official landing page repository for **KidWay**, a startup focused on school transport management solutions for independent operators, transport companies, schools, and families.

This repository contains the public-facing website built with **HTML, CSS, and JavaScript**, designed to present the business model, services, benefits, pricing plans, testimonials, FAQs, and contact channels.

---

## Purpose of This Repository
The objective of this project is to provide a professional and responsive landing page that:

- Introduces the KidWay value proposition.
- Explains how the platform works.
- Shows benefits for target segments.
- Presents pricing plans.
- Captures leads through contact forms.
- Redirects users to the main web application.
